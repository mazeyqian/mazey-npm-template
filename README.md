## Install

You can get mazey-npm-template via [npm](https://www.npmjs.com/package/mazey-npm-template).

```
npm install mazey-npm-template --save
```

## Usage

Foo description.

```
import { foo } from 'mazey-npm-template';

foo();
```

Bar description.

```
import { bar } from 'mazey-npm-template';

```
